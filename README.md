# 📚 Projeto Escola SENAI

Este projeto é uma página web desenvolvida em **HTML e CSS**, com o objetivo de apresentar informações sobre o SENAI, seus cursos e possibilitar o contato com o usuário através de um formulário.

## 📌 Sobre o Projeto

A aplicação consiste em um site simples e informativo contendo:

* Página principal com informações sobre o SENAI
* Página secundária ("Sobre") com mais detalhes da instituição
* Tabelas de cursos e notas
* Links para redes sociais
* Formulário de contato

O projeto foi desenvolvido com foco em prática de estrutura HTML e organização de conteúdo.

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3

## 📂 Estrutura do Projeto

```
📁 projeto-senai
│-- index.html
│-- sobre.html
│-- styles.css
```

## 🖥️ Funcionalidades

✔️ Navegação entre páginas
✔️ Exibição de informações institucionais
✔️ Tabela de cursos oferecidos
✔️ Tabela de notas de alunos (exemplo)
✔️ Links externos (Instagram, YouTube e site oficial)
✔️ Formulário de contato

## 📸 Layout

O site contém:

* Cabeçalho com navegação
* Imagens ilustrativas
* Seções organizadas com títulos e textos
* Tabelas informativas
* Área de contato

## ⚠️ Observações

* Algumas imagens utilizam links externos ou base64
* O formulário não possui integração com backend (apenas estrutura HTML)
* Pequenos ajustes podem ser feitos no código (como correção de atributos e organização de tags)
